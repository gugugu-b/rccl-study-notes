SendRecv · busBw 理论上限推导
源码 sendrecv.cu: factor = 1，n = 8 GPU，M = 128 MB
推导链
1M = 128 MB, n = 8每 rank 向 peer 发送 128MB，接收 128MB
2每 rank 经一条链路发送 M全双工下发送与接收可并行
3T = M / B单链路传输时间
4algBw = M / T = B算法带宽等于链路带宽
5busBw = algBw × 1 = B ✓factor=1，纯点对点无集合开销
busBw = B
SendRecv 是所有集合操作的原子基线，复杂集合可分解为多次 SendRecv 编排。与 Broadcast/Reduce 同属 factor=1 单链路瓶颈类。
